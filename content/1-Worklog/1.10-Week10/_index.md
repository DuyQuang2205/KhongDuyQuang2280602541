## Challenges Encountered

* Some issues only appeared after deployment to AWS and were not reproducible in the local environment.
* API optimization required balancing performance and maintainability.
* Several frontend issues occurred due to changes in backend responses.
* Comprehensive testing required preparing sufficient test data for different scenarios.

---

## Lessons Learned

* Continuous testing helps identify problems early and reduces debugging time later.
* Distinguishing whether an issue originates from the frontend, backend, or AWS infrastructure makes troubleshooting more efficient.
* Amazon CloudWatch is an effective tool for monitoring application logs and system performance.
* Deployment documentation should be updated throughout the development process.
* A successful system should provide not only complete functionality but also stability, good performance, and maintainability.
